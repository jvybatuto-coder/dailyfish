# Daily Fish - Fish Marketplace

Django-based fish marketplace application.

## Setup

1. Create virtual environment: `python -m venv myenv`
2. Activate: `myenv\Scripts\activate`
3. Install dependencies: `pip install -r myproject/requirements.txt`
4. Run migrations: `python myproject/manage.py migrate`
5. Start server: `python myproject/manage.py runserver`

## Features

- Fish listing and detail pages
- Shopping cart functionality
- User authentication
- Order management
- Review system
